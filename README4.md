# Basic Weather App

## Description
The Basic Weather App is a Python application that provides current weather information for a specified city. It fetches weather data from an online weather API and displays details such as temperature, humidity, weather conditions, and wind speed.

## Features
- Search weather by city name
- Display current temperature
- Show humidity percentage
- Display weather description
- Show wind speed
- Error handling for invalid city names
- Real-time weather updates

## Steps to Make It
1. Install the Requests library.
2. Get an API key from OpenWeatherMap.
3. Import required modules.
4. Accept city name input.
5. Send API request.
6. Process JSON response.
7. Display weather information.
8. Handle errors.
9. Test the application.

## Advantages
- Real-time weather information
- Easy to develop and understand
- Improves API integration skills
- Useful for learning JSON handling
- Can be extended with additional features
