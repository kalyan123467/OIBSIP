# BMI Calculator Project

## Description
The BMI (Body Mass Index) Calculator is a Python application that calculates a person's BMI using height and weight inputs and classifies health status.

## Features
- Calculates BMI accurately
- Accepts height and weight input
- Displays BMI instantly
- Health status classification
- User-friendly interface
- Input validation

## Steps to Make It
1. Import required libraries.
2. Take height and weight as input.
3. Apply the BMI formula:
   BMI = Weight (kg) / Height² (m²)
4. Classify the BMI result.
5. Display the BMI value and category.
6. Test the application.
